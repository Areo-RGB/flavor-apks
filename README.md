## Build and Install (Windows)

1. `npm run build:debug:apk`
2. `npm run install:debug:devices`
3. `npm run run:debug:devices`

## Native Test Command

`cd android && gradlew.bat :app:testDebugUnitTest`

## Notes

- This repository is Android-only.
